# Machine Fault Detection Using Sound Analysis

## Overview
This project detects machine faults by analyzing sound signals using digital signal processing and machine learning.  
The system classifies machine sounds into two categories: **Normal** and **Faulty**.

## Objective
- Detect machine faults using sound signals
- Build a simple and cost-effective monitoring system
- Use machine learning for classification

## Technologies Used
- Python
- Google Colab
- Librosa
- NumPy
- Matplotlib
- Scikit-learn

## Concepts Used
- Digital Signal Processing (DSP)
- MFCC (Mel Frequency Cepstral Coefficients)
- Supervised Machine Learning
- Random Forest Classifier
